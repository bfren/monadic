Match<T>(Maybe<T>, Action, Action<T>)

Run an action based on the value of maybe.

public static void Match<T>(Maybe<T> maybe, Action none, Action<T> some)

Parameters

maybe Maybe<T>

Maybe object.

none Action

Function to run when maybe is None.

some Action<T>

Function to run when maybe is Some<T>.

Type Parameters

T

Some value type.

Remarks

It would be possible to achieve the same thing using switch statements, but doing that would add the potential for not handling both cases (Some<T> and None).

This way there has to be a function to handle both cases.

Exceptions

InvalidMaybeTypeException
NullMaybeException
