MatchAsync<T>(Result<T>, Func<FailValue, Task>, Func<T, Task>)

Run an action based on the value of result.

public static Task MatchAsync<T>(this Result<T> @this, Func<FailValue, Task> fail, Func<T, Task> ok)

Parameters

this Result<T>
fail Func<FailValue, Task>

Function to run when result is Fail.

ok Func<T, Task>

Function to run when result is Ok<T>.

Returns

Task

Type Parameters

T

Ok value type.

Remarks

It would be possible to achieve the same thing using switch statements, but doing that would add the potential for not handling both cases (Ok<T> and Fail).

This way there has to be a function to handle both cases.
